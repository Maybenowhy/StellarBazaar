## ⚙️ 主要功能

### **用户端（Customer）**

-   浏览商品
-   按分类筛选
-   商品详情页
-   购物车（增删改）
-   提交订单
-   查询订单状态

### **管理端（Admin）**

-   商品管理（上架 / 下架 / 修改）
-   订单管理（处理 / 确认）
-   销售统计（折线图 & 表格）
-   用户列表管理
-   权限控制（管理员可访问所有后台功能）

## 🔑 角色权限说明

平台默认有两种角色：

  ----------------------------------------------------------------------------------------
  角色                   权限                           页面入口
  ---------------------- ------------------------------ ----------------------------------
  **customer（用户）**   浏览商品、购物车、订单         `/user/products`

  **sales（管理员）**    商品管理、订单管理、用户管理   `/sales/user-list`
  ----------------------------------------------------------------------------------------

登录后后端返回：

``` json
{
  "token": "...",
  "role": "customer" / "sales"
}
```

前端根据 `role` 自动跳转对应页面。

## 🤝 开发者须知

-   所有 API 均需携带 token
-   角色不同导致跳转路径不同
-   管理端和用户端本质为同一个前端项目中的不同模块
-   若无需用户端，可将其隐藏并仅保留管理员系统
